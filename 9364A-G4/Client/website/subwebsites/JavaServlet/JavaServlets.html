What are Java Servlets?

These are programs that run on Web or Application server and act as a middle layer between requests coming from a browser and databases or apps on the HTTP server. Java objects intended to play the role of server components in client-server communications.

Handles client requests by generating responses to such requests.

Part of the Java EE specification

    HTTP Servlets

  • Hosted in a web container which is a component in a java application server
  • service() method call is routed to a doXXX() call, depending on the HTTP request method
  • doXXX() methods has 2 arguments:
    • HTTPServletRequest Used to access information from the request message
    • HTTPServletResponse Used to set information in the response message
