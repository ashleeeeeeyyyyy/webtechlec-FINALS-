Php Tags
  • Php parses a file through looking for a Opening and Closing tags which are the
    <?php and ?>.These tags tells php that it will interpret the codes
    that are blocked in between them.This result for php to also not to parse the codes
    that are not in between those blocks.
  • Also if your file is a pure Php code,It is advicable to NOT the closing tag
    of the php to avoid white spaces to be added after the closing tag

  • Example: 
     <php 
                echo "Hello WBTKLC-Php!
                // ... insert Codes here
                echo "GoodBye WBTKLC-Php ));
                //The Script ends here with no php code
