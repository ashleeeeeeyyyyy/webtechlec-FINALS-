HTTP Servlets

  • Hosted in a web container which is a component in a java application server
  • service() method call is routed to a doXXX() call, depending on the HTTP request method
  • doXXX() methods has 2 arguments:
    • HTTPServletRequest Used to access information from the request message
    • HTTPServletResponse Used to set information in the response message
    A Few Examples:
    • arrow_drop_downdoGet()
       protected void doGet(HttpServletRequestreq, HttpServletResponse response)throws ServletException, java.io.IOException
                                      
                                      
    •  This is called by the server, using the service() method, to allow the servlet to handle a GET request. 
      • 
                                        
    •  This is usually used when a small amount of data like a query will be sent as a request.
    • arrow_drop_downgetLastModified()
       protected long getLastModified(HttpServletRequest req)
                                      
                                      
    • This returns the time the HttpServletRequest object was most recently modified in milliseconds.
      • 
                                      
    • If the time is unknown, it will return a negative number
      •
    • arrow_drop_downdoHead()
       protected void doHead(HttpServletRequest req,
                            HttpServletResponse resp)
                               throws ServletException,
                                      java.io.IOException
                                      
                                      
    •  This method is overriden to handle a HEAD request. 
      • 
                                
                                      
    •  This only returns the header information same as a get request would do,but without the entity body
      •
    • arrow_drop_downdoPost()
       protected void doPost(HttpServletRequest req,
                      HttpServletResponse resp)
               throws ServletException,
                      java.io.IOException
                                      
                                      
    •  This is called by the service() method to allow the servlet to handle a POST method. 
      • 
                                
                                      
    •  The POST method allows the cient to send data with an unlimited length to the server.
      • 
                                      
                                      
    •  This can also be used to actually upload files such as long text files and the like.
      •
    • arrow_drop_downdoPut()
