Java Servlets

These are programs that run on Web or Application server and act as a middle layer between requests coming from a browser and databases or apps on the HTTP server. Java objects intended to play the role of server components in client-server communications. This handles client requests by generating responses to such requests.It is also a part of the Java EE specification.

ADVANTAGES

Good impressions often give popularity to a certain something. Knowing what makes servlets makes the user more interested to the subject.

Advantages

  • Its performance is slightly increased

  • Servlets are platform-independent, reason being that Java Servlets are run and written in Java.

  • Servlets are executed wothin the address of a Web Server. It is unnecessary to create separate processes to handle every request.

  • Servlets can communicate with other servlets and servers.

  • Servlets enable easy portability across different web servers.

  • Servlets provide all feautures of Java like Exception handling and so.
