1. This requires a constructor call which is then invoked implicitl by the servlet container.

a. Request Handling b. Initialization c. Instantiation

correct answer: c. initialization


2. The Request Handling phase of a servlet lifecycle utilizes the _______ method.

a. destroy() b. service() c. init()

correct answer: b. service()


3. Complete the code: public void init(_______ conf) throws ServletException

a. ServletRequest b. ServletConfig c. ServletInitialize

correct answer: b. ServletConfig


4. What stage processes the client's request?

a. Destruction b. Instantiation c. Request Handling

correct answer: c. Request Handling


5. This is used to access information from the request message.

a. HTTPServletRequest b. HTTPServletResponse c. HTTPServletMessage

correct answer: a. HTTPServletRequest
